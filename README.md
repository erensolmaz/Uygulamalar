# Çoklu Uygulamalar

## Description
> Birçok uygulamanın bir arada olduğu java ile yazılımş mobil app.

### Features
- Login
- Calculator
- Currency Calculator
- Listview
- Dark and light themes
- Galery
- Bottom navigation
- Pop-up messages
